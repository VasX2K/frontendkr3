О приложении

Версия 1.2.3

Это приложение для заметок с поддержкой офлайн-режима, push-уведомлений и напоминаний.

Оно использует Service Worker для кэширования ресурсов, Web App Manifest для установки на устройство, WebSocket для обмена данными в реальном времени и Push API для уведомлений.

Возможности:

  • Создание заметок с сохранением в localStorage
  • Работа в офлайн-режиме
  • Установка на устройство как PWA
  • Push-уведомления о новых задачах
  • Напоминания с возможностью отложить
  • Синхронизация через WebSocket
